data Configuration

Settings like application available port or static files location are represented by `Configuration`. Instead of using (String,String) pairs, it seems to be safer to use types to represent those settings, e.g instead of writing ("port", "808x") and receive a runtime error it's better to write `port 8080` which is checked at compile time.
